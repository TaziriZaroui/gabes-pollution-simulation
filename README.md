# Gabès Pollution Impact Simulation
 ***#Gabès Through the Lens of Simulation***
 
****Simuler l'Impact | Sauver des Vies | Protéger Gabès
"Il était une fois, Gabès était un joyau méditerranéen..."
Ce projet n'est pas qu'une simple simulation statistique. C'est un témoignage numérique qui quantifie la douleur silencieuse d'une communauté, la perte invisible d'années de vie, et le prix humain du "développement" industriel.****

Résumé
Problème : Pollution industrielle à Gabès-Ghannouch
Méthode : Simulation basée sur les modèles de risque OMS
Résultat : Jusqu'à 15,7 ans d'espérance de vie perdus dans le pire scénario

Objectifs
Quantifier les années de vie potentielles perdues

Visualiser l'impact sanitaire invisible

Soutenir des politiques environnementales

Sensibiliser aux risques de pollution

PM2.5 : Le Danger Invisible
Définition : Particules fines (< 2.5 μm) émises par l'industrie du phosphate

Impacts :

Pénétration pulmonaire profonde

Passage dans la circulation sanguine

Accumulation sur plusieurs années

Affecte cœur, poumons, cerveau

Modèle Mathématique
text
LE_adj = LE_ref / exp(β × ΔPM2.5)
Où :

LE_adj = Espérance de vie ajustée

LE_ref = 77.0 ans (Tunisie 2023)

β = 0.0077 (coefficient OMS)

ΔPM2.5 = Augmentation PM2.5

Source : OMS (2021) Global Air Quality Guidelines

Note Méthodologique
Ces valeurs sont des projections hypothétiques basées sur :

La tendance actuelle d'augmentation de la pollution

Les niveaux observés dans des zones industrielles similaires

Une projection de ce qui pourrait arriver sans intervention

Nous ne disposons pas de mesures continues officielles des PM2.5 à Gabès.
